# Payment API Test Project

This project is a simple test suite for **PayPal** and **Stripe** APIs using **fake accounts** and mock calls.

## Features

- Simulates online payments with test credit cards.
- Covers common payment scenarios:
  - Successful payment.
  - Insufficient funds.
  - Card declined or unknown card.

## Disclaimer

- This project **does not perform real transactions**.
- It is intended **for development and testing purposes only** using sandbox environments and mocked responses.
